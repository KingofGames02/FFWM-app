## English

### 📝 Description
**Wishlist Manager** is a high-performance web tool designed for players to manage their game wishlists with ease and precision. The tool allows you to log in securely using game session files (`.dat`) or manual credentials, providing a seamless interface to control your in-game items and explore others' profiles.

### ✨ Key Features
* **Flexible Login Options:** Securely log in by uploading your `.dat` session file or by entering your UID and Password manually.
* **Bulk Item Management:** Add or remove multiple items at once (Bulk) with a real-time progress bar to track the process.
* **Advanced Player Discovery:** Search for any player by UID or Nickname to view their wishlist and copy item IDs directly to your own list.
* **Item Library Integration:** Directly linked with the **Free Fire Items Library** for quick access to the latest item IDs and rarities.
* **Smart Bilingual UI:** A sleek, modern interface that supports both English and Arabic, switchable with a single click.
* **Fully Responsive:** Optimized for a perfect experience across mobile devices, tablets, and desktops.

## العربية

### 📝 الوصف
**مدير قائمة الأمنيات (Wishlist Manager)** هو أداة ويب متطورة مصممة لمساعدة اللاعبين في إدارة قوائم أمنياتهم داخل اللعبة بكل سهولة ودقة. تتيح لك الأداة تسجيل الدخول بأمان عبر ملفات الجلسة (`.dat`) أو البيانات اليدوية، وتوفر واجهة احترافية للتحكم في عناصرك واستكشاف ملفات اللاعبين الآخرين.

### ✨ المميزات الرئيسية
* **خيارات تسجيل دخول مرنة:** دعم كامل لتسجيل الدخول عبر رفع ملف الجلسة (`.dat`) أو إدخال المعرف (UID) وكلمة المرور يدوياً.
* **إدارة قائمة أمنيات الحساب:** يمكنك إضافة أو إزالة عدة عناصر في وقت واحد مع شريط تقدم حي لمتابعة حالة العملية لحظة بلحظة.
* **استكشاف اللاعبين:** ابحث عن أي لاعب عبر المعرف أو الاسم لتصفح قائمة أمنياته ونسخ معرفات العناصر (IDs) منها مباشرة.
* **التكامل مع مكتبة العناصر:** الأداة مرتبطة بمكتبة **Free Fire Items Library** لسهولة الوصول إلى أحدث العناصر وتصنيفاتها.
* **واجهة ثنائية اللغة:** تصميم عصري يدعم اللغتين العربية والإنجليزية بشكل كامل مع سهولة التبديل بينهما.
* **توافق تام مع جميع الشاشات:** تجربة مستخدم سلسة ومحسنة للعمل على الهواتف، الأجهزة اللوحية، وأجهزة الكمبيوتر.

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (Hosted on GitHub Pages).
* **Backend:** Python (Flask), deployed on Vercel.
* **Protocol:** Protobuf for handling secure game data communication.
